# Node Hono API

## Setup

Copy the env from `.env.example` by running the command

```
cp .env.example.env
```

Setup apporpriate environment variables

Use pnpm for package management

```
pnpm install
pnpm run dev
```

open `http://localhost:<PORT>`

## Stack

- NodeJS
- HonoJS - lightweight API library
- Zod - input validations
- MongoDB - database
- Jest - testing
